Tag <h1> a <h6>

Essas tags servem pra criar títulos de diferentes tamanhos e importância. O <h1> é o título principal, geralmente o maior e mais importante da página. À medida que o número aumenta, o tamanho do texto vai diminuindo, até o <h6>, que é o menor. É uma forma de organizar melhor a hierarquia das informações no conteúdo. Por exemplo, você pode usar um <h2> para subtítulos dentro do assunto principal do <h1>.

Tag <p>

A tag <p> é usada para escrever parágrafos, ou seja, blocos de texto com começo, meio e fim. Sempre que você quiser colocar um texto mais explicativo, contar algo ou simplesmente descrever uma informação, use o <p>. Além disso, ela já vem com uma pequena margem automática entre os parágrafos, o que ajuda na leitura.

Tag <mark>

Essa tag é como se fosse um marca-texto digital. Serve pra destacar uma palavra ou frase importante dentro do texto. Por exemplo, você pode marcar uma informação essencial para que ela chame a atenção do leitor, como essa aqui.

Tag <small>

Quando você quiser colocar uma observação ou nota de rodapé em tamanho menor do que o texto normal, essa é a tag certa. É muito usada pra avisos, detalhes legais ou complementos que não precisam aparecer com tanto destaque, tipo aquele “*termos e condições se aplicam”.

Tag <i>

Essa tag serve pra deixar o texto em itálico. Mas não é só questão de estilo: ela é usada pra mostrar que aquele trecho tem um significado especial. Pode ser um termo estrangeiro, uma expressão diferente ou uma palavra usada com ênfase.

Tag <u>

Ela sublinha o texto, e hoje em dia costuma ser usada pra indicar que aquela parte tem um destaque ou uma anotação importante. Antigamente era comum sublinhar pra dar ênfase, mas hoje essa função pode ser feita melhor com CSS. Mesmo assim, a tag ainda funciona.

Tag <strong>

É usada quando você quer dar uma importância maior pra um trecho. O navegador normalmente mostra esse texto em negrito, mas o principal aqui é o significado: aquilo que está entre <strong> é forte, relevante, sério ou urgente.

Tag <ol>

Essa tag cria uma lista numerada. Cada item da lista é exibido com um número na frente. Isso é ótimo pra mostrar sequências, passo a passo, ou qualquer coisa que tenha uma ordem definida, tipo: 1º passo, 2º passo, 3º passo.

Tag <ul>

Com essa tag você cria uma lista com marcadores, sem ordem definida. Serve pra listar coisas soltas, como ingredientes de uma receita ou tópicos de estudo, por exemplo. Os itens são marcados com bolinhas (ou outros símbolos, dependendo do navegador).

Tag <li>

Essa tag representa cada item dentro de uma lista, seja ela ordenada (<ol>) ou não ordenada (<ul>). Sempre que for colocar algo dentro dessas listas, deve usar a tag <li> para separar os elementos.

Tag <a>

Ela é usada pra criar links. Com o atributo href, você define pra onde o link vai levar, como uma outra página, um site externo, um e-mail ou até uma parte da própria página. É uma das tags mais importantes da web, afinal, a internet é feita de links!

Tag <hr>

Insere uma linha horizontal que divide seções de conteúdo. É uma forma visual de mostrar uma mudança de assunto ou uma separação entre partes do conteúdo.

Tag <sub>

Essa tag coloca o texto um pouco abaixo da linha normal, em tamanho menor. É muito usada em fórmulas químicas, como H2O, ou pra indicar rodapés ou notas pequenas.

Tag <sup>

Coloca o texto um pouco acima da linha normal, também com fonte menor. É ideal pra coisas como x2 (potência), notas de rodapé ou referências.

Tag <blockquote>

Serve pra mostrar que aquele trecho é uma citação longa, tirada de outro lugar. O navegador geralmente já mostra essa citação com recuo, pra ficar bem destacado do resto do texto.

Tag <font>

Antigamente, essa tag era usada pra mudar o tipo, a cor e o tamanho da fonte. Hoje em dia ela está em desuso, pois o ideal é fazer isso com CSS. Mesmo assim, ainda funciona em alguns navegadores.

Tag <del>

Indica que aquele texto foi removido ou substituído, como se fosse um “risco” em cima. É útil pra mostrar que houve uma edição ou correção. Exemplo: Texto antigo.

Tag <abbr>

Essa tag mostra que aquela palavra é uma abreviação. Você pode usar o atributo title pra mostrar o significado completo ao passar o mouse. Por exemplo, CSS.
